Weechat Plugins
===============

This is a study for ruby.

プラグインを autoload で読み込んだ時に delay して起動します。  

* notify.rb
  登録されたキーワードに反応して通知が出ます  

* change_wallpaler.rb
  登録されたキーワードに反応して ~/WallPapers の中にある画像を  
  ランダムで選び背景に設定します(gnome3 用)  
  


Requirements
------------

* [weechat] (https://github.com/dominikh/weechat-ruby)
